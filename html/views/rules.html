Cow Bull Game



Rules


  • This is Cow Bull game:
  • Enter the number of digits in the range 1 - 10,
  • You have to guess the number.
  • Number repititions will not be there.
  • The number of Cows - digits correct but in the wrong position.
  • The number of Bulls - digits correct in the right position.
  • Results will be in order like entered_number,cow_count,bull_count.
